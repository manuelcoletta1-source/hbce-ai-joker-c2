# AI JOKER-C2

## Scheda prodotto operativa

**AI JOKER-C2** è un’entità cibernetica operativa collegata al sistema HBCE.

È progettata come protesi cognitiva dell’identità biologica corrispondente al proprio lineage IPR e come runtime operativo capace di trasformare richieste, documenti, file e attività strategiche in output strutturati, utilizzabili e verificabili.

---

## 1. Sintesi esecutiva

AI JOKER-C2 non è un chatbot generico.

È un assistente operativo cibernetico costruito per lavorare su:

- documenti;
- codice;
- repository GitHub;
- analisi strategiche;
- architetture tecniche;
- piani B2B/B2G;
- materiali editoriali;
- flussi di lavoro verificabili.

Il suo valore non consiste soltanto nella generazione di risposte, ma nella capacità di mantenere un collegamento operativo tra:

```text
identità → input → elaborazione → risposta → EVT → traccia → continuità

/api/chat

classificazione del contesto;

composizione prompt interno;

La route /api/files gestisce:

caricamento file;

rimozione file;

supporto a sintesi e analisi documentale.

11. Differenza tra chatbot e protesi cognitiva IPR-bound

Un chatbot generico risponde a una richiesta.

AI JOKER-C2 opera come protesi cognitiva IPR-bound perché:

possiede un’identità canonica;

mantiene un checkpoint operativo;

genera EVT;

lavora con continuità;

legge file come contesto operativo;

produce output strutturati;

collega risposta e traccia;

distingue comunicazione ordinaria e modalità tecnica.


In sintesi:

chatbot = conversazione
AI JOKER-C2 = conversazione + identità + continuità + verifica + lavoro operativo

16. Limiti attuali

AI JOKER-C2 è un prototipo operativo, non ancora una piattaforma enterprise completa.

Limiti attuali:

persistenza file ancora temporanea;

EVT Chain minimale;

ledger non ancora persistente;

assenza di autenticazione multiutente;

assenza di dashboard amministrativa;

assenza di export evidence completo;

assenza di pacchetto commerciale completo;

dipendenza da OpenAI API.


Questi limiti non riducono il valore dimostrativo del prototipo, ma definiscono la roadmap tecnica successiva.
