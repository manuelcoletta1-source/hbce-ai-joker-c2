## Introduzione

L'Identity Primary Record (IPR) rappresenta un identificatore persistente associato a una entità operativa.

L'obiettivo dell'IPR è consentire l'attribuzione delle azioni digitali nei sistemi tecnologici complessi.

---

## Entità supportate

Un IPR può essere associato a:

- operatori umani
- agenti AI
- robot
- droni
- infrastrutture digitali
- servizi software

---

## Funzioni principali

L'IPR consente:

- identificazione operativa
- attribuzione delle azioni
- tracciabilità delle operazioni
- integrazione con registri di eventi
