# app

Application runtime layer of AI JOKER-C2.

This directory contains the user-facing application surface of the HBCE AI JOKER-C2 system.

Its role is to provide the operational interface through which users interact with the underlying HBCE infrastructure.

## Scope

The application layer may include:

- conversational interface
- operational request composer
- runtime status panels
- identity and verification display
- interaction flow with the governance pipeline

## Position in the System

The `app` layer connects:

user interaction  
→ application runtime  
→ execution gateway  
→ HBCE governance infrastructure
